# 🧩 Interactive Product Showcase (Next.js + TailwindCss + Framer motion)

## 📌 Overview

This project is an interactive product showcase built with **Next.js**, **TypeScript**, **JavaScript**, and **Tailwind CSS**. It features animated hover cards that display product details when interacting with geometric elements.

---

## 🚀 Features

* ⚡ Next.js for structure and performance
* 🎨 Tailwind CSS for styling
* 🎬 Framer Motion for animations
* 🧠 Hover-based interactions
* 🔁 Dynamic rendering using `.map()`
* 🌐 API data fetching with custom hook

---

## 🏗️ Project Structure

```
/components
  ├── Heptagon.tsx
  ├── HoverCard.tsx
  ├── Navbar.tsx
  └── Footer.tsx

/pages
  ├── HomePage.tsx
  └── ProductPage.tsx

/hooks
  └── useApiData.ts
```

---

## ⚙️ Approach

* Used **component-based design** for clean structure
* Implemented **dynamic UI rendering** with mapped data
* Managed hover state for interactive behavior
* Added **smooth animations** using Framer Motion
* Separated API logic with a reusable custom hook
